Part 2: Code up your flow chart (solution)

Step 1. Write winner_RPS

Create a function named winner_RPS. The function takes two parameters, p1 and p2.

  • It returns p1 (the parameter) if player1 wins; p2 if player2 wins; and None if there is no winner.
  • Write your solution such that the winner_RPS has a single return statement.
  • p1 and p2 can only be one of the following values: "rock", "paper", "scissors"
