The Console; the Terminal and REPL

Many programming environments call the combination of where you write and execute commands the terminal or console (not to comfort someone, nor a gaming system, nor your grandfather's tv cabinet). Traditionally, when you write code in a console/terminal, the Python interpreter will Read each statement, the statement is Evaluated and the result is Printed and the process starts over again (aka a Loop).

A better calculator

So far, at the very least, we have a simple calculator. Let's try another (see if you can get the answer before running it):

5 * 2+3

The answer, 13, might seem strange if you were expecting 25. The interpreter has rules on how to evaluate mathematical expressions. To get 25, type the following in the next code cell:

5 * (2+3)

Programming languages put priority on evaluating things in parenthesis first. We'll have more to say about operators (symbols used to perform operations like + for addition, * for multiplication, / for division) and operator precedence (the order in which they are evaluated) in later lessons.
